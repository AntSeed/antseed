# Global Flags

```bash title="flags"
-c, --config <path>     Path to config file (default: ~/.antseed/config.json)
--data-dir <path>       Path to node identity/state directory (default: ~/.antseed)
-v, --verbose            Enable verbose logging
--version                Show version
--help                   Show help
```
